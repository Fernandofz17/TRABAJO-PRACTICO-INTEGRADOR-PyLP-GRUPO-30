### Descripción
Implementación de un Sistema de Semáforos Inteligentes utilizando Common Lisp, aplicando conceptos de programación funcional, inmutabilidad y composición funcional.

### Estructura del repositorio

- `lisp/`: implementación principal en Common Lisp.
- `comparativa/`: reimplementación de funciones en OCaml.
- `docs/`: documentación e informe del proyecto.

### Tecnologías utilizadas

- Common Lisp
- SBCL
- Quicklisp
- CL-JSON
- OCaml
